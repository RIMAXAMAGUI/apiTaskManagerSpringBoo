Información de la api

Crearemos una aplicación web (Api rest) que permita la gestión de tareas personales, y de cada tarea se deberá guardar:
- Titulo
- Descripción
- Fecha creación
- Fecha estimada de terminación
- Fecha Finalización
- Finalizada (si/no)
- Estado (on time / late)

La funcionalidad que necesitamos son:
1- Crear tareas
2- Obtener todas las tareas
3- Obtener todas las tareas filtradas por estado
4- Marcar tarea como cumplida (actualizar tarea)
5- Eliminar tarea

Usaremos Spring boot, H2 & Swagger para crear esta aplicación.
